## 點解咁排

1. **時間軸 + 即時 AI** 係 tipsme 式「開場就有得睇」嘅核心差異——無事件就無分析素材。  
2. **2D 球場** 視覺吸睛，但依賴事件／統計，所以同 P0 一齊上 MVP。  
3. **聊天／投票** 提升留存，但唔影響「分析產品」可信度，可平行。  
4. **盤路 API** 要牌照／付費源，先手動 + 結構預留。  
5. **H2H** 難度低，ESPN 有資料，S2 補齊 UI。
